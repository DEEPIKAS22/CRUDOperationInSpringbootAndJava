## crud-spring-boot-thymeleaf
 Simple crud application to perform basic tasks, that we have to know while learning spring boot.


## Technolgies Used 
  1. Spring-boot 3.0.0
  2. Thymeleaf template engine
  3. Spring data jpa
  4. Spring mvc
  5. MySQL database
  6. Developed in Eclipse EE

Installation

    Clone the Repo:
    > git clone https://github.com/ProjectsAndPrograms/Ideas.git
    > cd Ideas
    > composer install or composer update
    > cp .env.example .env
    > Set up .env file
    > php artisan key:generate
    > php artisan storage:link
    > php artisan migrate:fresh --seed
    > php artisan serve
    http://127.0.0.1:8000/

<br>
setup your .env file correctly it contains database credentials
